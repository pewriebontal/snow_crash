# so_long (SNOW CRASH)

## Description
I named this game SNOW CRASH. The lore is inspired by the book "Snow Crash" by Neal Stephenson. It serves as my submission for the **so_long** project, a basic 2D game requiring texture mapping, valid map parsing, and window management via MiniLibX.

## Instructions

### Compilation
Compile the project using the provided `Makefile`:
```bash
make
```

### Execution
Run the game using a valid `.ber` map file:
```bash
./so_long maps/valids/simple_valid.ber
```

### Controls
* **W, A, S, D:** Move up, left, down, and right, or you may also use the arrow keys for movement.
* **ESC, Q:** Close the game.
* **Window UI:** Click the close button/cross to exit.
* **]** Toggle the exit door's open/close state (for testing purposes).
